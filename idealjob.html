Ideal Job

What is this job?

A role based around developing programs, applications and other software using GIS, a spatial/geographical based software. The projects involve working on previously established work, which includes maintenance and optimisation of in-house programs. This role requires knowledge in GIS/Spatial, Python/Java, Javascript (Node/React), Postgre SQL, and finds AWS API GAtways knowledge also desirable. Outside of programming knowledge, this requires key job components such as problem solving and communication skills.

What do I have? What do I need?

Of these requirements, I have already gathered problem solving and communication skills from many various roles I have worked in the past, and expect to be able to practice and keep these skills in the future.

Of the requirements I currently don’t posses, namely the coding/programming skills, I’d hope to be able to learn, expand on and experience all of these through university study, placements and future job opportunities in the IT pathway.
